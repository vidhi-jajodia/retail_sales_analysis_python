# 🛒 Retail Sales Analytics Dashboard

An end-to-end data analytics project built with Python and Streamlit to analyze retail sales, customer behavior, and profitability.

---

## 🌐 Live Demo

🚀 *[https://retail-sales-analysis-python.streamlit.app/]*

---

## 📊 Project Overview

This project performs:

* Data cleaning & preprocessing
* Exploratory Data Analysis (EDA)
* Customer segmentation using RFM analysis
* Interactive dashboard with filters and visualizations
* AI-generated insights for decision-making

---

## ✨ Features

### 📊 Interactive Dashboard

* Filter by **Region, Category, Date Range**
* Dynamic charts with **hover, zoom, and drill-down**

### 🧠 AI-Generated Insights

* Automatically explains:

  * Top-performing categories
  * Regional performance
  * Loss-making products
  * Discount impact on profit

### 📈 Visualizations

* Sales by Category
* Profit by Sub-Category
* Monthly Sales Trend
* Discount vs Profit
* Geographic Sales Map

### 👥 Customer Segmentation

* RFM Analysis (Recency, Frequency, Monetary)
* Customer grouping:

  * Champions
  * Loyal Customers
  * Big Spenders
  * At Risk

### 📥 Export Options

* Download filtered data (CSV)
* Export RFM analysis

### 🌙 UI Enhancements

* Light / Dark mode toggle
* Clean BI-style layout

## ⚙️ Installation & Setup

### 1. Clone the repository

```bash
git clone https://github.com/kumawat-sanjay/retail-sales-analysis-python.git
cd retail-sales-analysis
```

### 2. Install dependencies

```bash
pip install -r requirements.txt
```

### 3. Run Streamlit app

```bash
streamlit run python/StreamlitApp.py
```
